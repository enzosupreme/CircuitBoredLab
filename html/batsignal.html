Bat Signal

Whenever I'm at home, I don't like being distracted by my phone. Though I stay away from my phone, and like to keep it on silent, I needed a non-invasive way to see when I get notifications. The idea was simple. Use a 3D-Printed Batsignal for phones to notify me while working at my desk.

Stage 1: Planning and Prep

The next step is the find a way to figure out how to get an LED to remotely turn off and on. After some research, I concluded that a wifi capable microcontroller was neccessary. The AdaFruit Feather Huzzah ESP8266 became the board of choice. Its a simple to use microcontroller with an ESP8266 chip giving it wifi capabilities.

Stage 2: Build

The part printed with no complications. I printed it without scaling it with 30% infill and a layer height of 0.2mm. Soldering the LED to the board was simple enough. Just two connections: one to ground and the other to a digital io pin.

Stage 3: Code

Setting up the code for this involved setting an ifttt applet. IFTTT, or "If This Then That", provided an easy platform to program a response to an android notification. Before creating an applet, I set up a feed using AdafruitIo, Adafruit's interface for connecting iot devices. After an empty feed was setup, I was able to make an ifttt applet that updates the feed everytime my android phone receives a notification by using the IFTTT app. I then used Arduino IDE to write a quick program that connects the microcontroller to wifi and checks the feed for updates. After it recieves the appropriate update it turns on the LED.
