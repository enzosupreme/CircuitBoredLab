Cyber Punkin

I wanted to make a cool Halloween decoration so using a color sensing module, I came up with a color changing jack-o-lantern. I call it the cyber punkin.

Design/Build/Code

I'm still not very good at 3d design so I started this project by following a great tutorial by "3D printy". I learned a lot watching this video and was able to design a few pumpkins of my own. After coming up with basic jack-o-lantern, I began wiring everytthing up. For this I used an itsy bitsy m4 express. I wired up an rgb led strip and a color sensing module. I used a AAA battery pack to power it up. While designing the pumpkin I made the stem on a seperate body to be printed in a variety of colors. This project didn't require too much code or wiring, I wrote a basic program that uses the libraries for the color sensing module to grab the RGB value that the senor detects then assign that value to the color of the RGB lights. I did run into the problem of not being able to figure how to deactivate the sensor. The original idea was to have a button that activates the sensor then store the color value in the RBG lights until the senor gets reactivated. Nothing I tried seemed to work so I came up with the idea of having the sensor peek through the top of the jack-o-lantern and have it detect the color of the steam.
