Midi Controller

Back in the before times, when I used to make music in my bedroom, I always wanted to make my own midi controller , but I lacked the technical knowhow and confidence. Well now the time has finally come.

Design/Build/Code

I made a midi controller inspired by the Midi Fighter. I made a compact midi controller with arcade style buttons with built in LEDs. I used a rotary encoder with led indicators to cycle through pitch while interfacing with a DAW such as Ableton. The enclosure was designed and printed by me. I used circuit python to write the code for this and everything went shockingly smooth. It works perfectly and I've already started jamming with it on a regular basis.
