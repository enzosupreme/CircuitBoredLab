PECS Board

I have a niece who is developmentally disable and has trouble speaking. I decided to build a PECS Board as a way to help her communicate.

Stage 1: Design

The initial challenge was figuring out how to design a PECS Board. I have plenty of key switches so, yep I made another project using key switches. First I designed a basic box enclosure, then made a flat plate to hold six keyswitches. I designed the enclosure with a hole for the power switch and one for the battery holder. I went with AA batteries to power it so that they would be easily replacable. I made small plates that would slide over each switch and house a flate plank the I could adhere the pictures to.

Stage 2: Build

I used a Circuit Playground Express since there was similar code written for it on adafruit's website. I started by fitting the keyswitches into the top plate. I then wired them all and grounded them to the Circuit Playground. The speakers were next, following a guide online, I wired up the speakers and a potentiometer to an amp, then connected to the dev board. I wired up a battery holder and the power switch and it was time to start on the code. Before being able to test it, i need some audio files. I used Audacity, a free recording/ audio editing software to record some short voice lines. After some recording and editing I had what I needed. Using Circuit Playground I set up each switch to make a call to function using two arguments, one for the audio file and one for color. I imported the same color pallette I made for my other projects, I imported those colors into a list to be called using the corresponding argument during the function.
