MagroPad

This one was short and sweet. I had some left over parts from my keyboard build and decided to make a small macropad with a twist. I embedded magnets into the print, because I thought it would be cool to be able to snap the macropad to the front of my PC case. I haven't added many features yet, but I wanted an interface for programmable buttons for whatever shortcuts I feel I wrote everything in circuit python. The MagroPad is set up with 3 separate layers that can be toggled by scrolling with the knob. The code for this was straight forward, I borrowed what I wrote for the GMShark giving me a foundation to write conditionals within each layer. The only functionality the MagroPad has for now are media controls. Layer one has prev track, pause/play and next track, the other layers control scroll and volume.
